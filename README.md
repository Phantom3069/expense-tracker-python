Expense Tracker

A console-based Expense Tracker built with Python.

Features:
- Add new expenses with category and date
- View all expenses
- Filter expenses by category
- View total and monthly summary
- Data saved to file

How to Run:
python expense_tracker.py

Tech Used:
- Python 3
- File Handling
- Date & Time module
